# Leo Wei -- Personal Portfolio

A gamified 3D portfolio website featuring an interactive developer workspace built with React and Three.js.

## Features

- **3D Dev Room** -- An interactive low-poly office scene with clickable objects that navigate to portfolio sections
- **Chibi Avatar** -- A waving character that greets visitors
- **Achievement System** -- 10 unlockable achievements and hidden easter eggs rewarding exploration
- **Day/Night Toggle** -- Click the window to switch between day and night
- **Interactive Objects** -- Growable plant, toggleable lamp, coffee mug, and more

## Tech Stack

- React 19 + Vite 6
- Three.js + @react-three/fiber + @react-three/drei
- Tailwind CSS v4

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```

Deploys to GitHub Pages via the `gh-pages` branch.
